# Risk-Factors-for-the-Development-of-Diabetes
Projet d’analyse et de visualisation de données explorant les principaux facteurs associés au développement du diabète. Ce dépôt comprend le traitement des données (pandas), l’analyse statistique et des représentations graphiques (matplotlib.pyplot) afin de mieux comprendre les profils de risque et les tendances liées au diabète.
